General Installation Instructions

1 Download and Extract

After downloading the extension ZIP file of your choice from Google Drive, extract its contents to a folder on your computer. Remember the location where you extracted the files.

2 Open Chrome Extensions Page

Open Google Chrome and navigate to the Extensions page using one of these methods:

  • Type chrome://extensions in the address bar and press Enter
  • Click the three dots menu (â‹®) in the top-right corner â†’ More tools â†’ Extensions

3 Enable Developer Mode

In the Extensions page, find the "Developer mode" toggle switch in the top-right corner and turn it ON. This will reveal additional developer options needed for installing unpacked extensions.

4 Load Unpacked Extension

Click the "Load unpacked" button that appears after enabling Developer Mode. This will open a file browser dialog.

Navigate to the folder where you extracted the extension files, select that folder (do not open it, just select it), and click "Select Folder" or "Open".

5 Verify Installation

After loading the extension, it should appear in your list of installed extensions with its icon and details. Make sure it shows as "Enabled".

If the extension doesn't appear, verify you selected the correct folder containing the extension's manifest.json file.

6 Pin to Toolbar (Optional)

To make the extension easily accessible, click the puzzle piece icon in Chrome's toolbar, find your newly installed extension, and click the pin icon next to it.
